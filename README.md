# Panqueco Café ☕

Projeto de uma landing page para cafeteria, desenvolvido com HTML e CSS.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3

## 📌 Funcionalidades
- Navegação por seções
- Layout responsivo básico
- Seção sobre a cafeteria
- Cardápio com produtos
- Avaliações de clientes
- Endereço com mapa incorporado
- Footer com redes sociais

## 💻 Objetivo do projeto
Este projeto foi desenvolvido para praticar estruturação de páginas com HTML, estilização com CSS e organização de layout para portfólio front-end.

## 📂 Estrutura do projeto
- `index.html`
- `style.css`
- `images/`
